📘 Facebook 粉專設定

📖 如何取得粉絲專頁存取權杖？（點此展開）
  1. 前往 Meta for Developers → Graph API Explorer
    網址：developers.facebook.com/tools/explorer
  2. 右上角選擇你的 Meta App
  3. 點「Add a Permission」，勾選以下三個權限：
    • pages_show_list
    • pages_read_engagement
    • pages_manage_posts
  4. 點「Generate Access Token」，完成授權
  5. 上方選單從「User Token」切換為你的 粉絲專頁名稱 → 複製下方 Token
  6. 將 Token 貼入下方欄位，點「✅ 驗證並載入」

⏰ Page Access Token 有效期約 60 天，到期後重新執行上述步驟即可。

Token 只存在你的瀏覽器，不會上傳。有效期約 60 天，到期再重新取得。
